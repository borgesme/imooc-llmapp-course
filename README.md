## python 虚拟环境

```bash
cd ./llmops-api

# 安装虚拟环境
python -m vnev env

# 激活虚拟环境
.\env\Scripts\activate

# 退出虚拟环境
.\env\Scripts\deactivate.

# 安装包
pip install openai
pip install langchain

# 更新镜像源 腾讯
pip config set global.index-url https://mirrors.cloud.tencent.com/pypi/simple/

# 查询验证
pip config list
```
